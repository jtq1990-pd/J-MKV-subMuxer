# J-MKV-subMuxer

本版本提供 macOS DMG 安装包，用于把视频文件和外部字幕文件封装成一个 MKV 文件。界面运行在 App 自己的 WKWebView 容器中，不依赖默认浏览器。

## 文件

- `J-MKV-subMuxer.dmg`

## 功能

- 添加单个任务：选择一个视频文件，再选择该视频的一组字幕文件。
- 添加文件夹：批量识别“1 个视频 + 一组字幕”为一个任务。
- 输出 `.mkv` 文件，默认后缀为 `_mkv封装.mkv`。
- 默认不重编码视频和音频。
- 可选择保留原视频内置字幕。
- 可设置外部字幕语言、轨道标题、默认字幕轨。
- 使用 App 内置 `mkvmerge` 完成封装。
- App 内置 `ffmpeg` 和 `ffprobe`，用于测试样本生成和后续扩展。
- `mkvmerge` 返回退出码 `1` 且输出文件有效时，任务标为“完成-有警告”，不再误判为失败。
- 已加入正式 App 图标，安装后在 Finder、Dock 和启动器中显示。
- v0.1.3 重新收敛主界面为“添加任务 -> 队列封装 -> 完成后清理”三段流程；环境信息改为可悬停标签；添加任务仅保留“批量添加文件夹”和“添加单个视频”两种方式；新增二次确认的“只保留封装视频”清理动作。

## 支持的输入扩展名

视频：

- `.mp4`
- `.mkv`
- `.mov`
- `.m4v`
- `.avi`
- `.webm`
- `.ts`
- `.m2ts`

字幕：

- `.srt`
- `.ass`
- `.ssa`
- `.sup`
- `.sub`
- `.idx`
- `.vtt`

`.idx` 字幕需要同目录存在同名 `.sub` 文件。

## macOS 依赖

DMG 内置运行所需的 `mkvmerge`、`ffmpeg`、`ffprobe` 和动态库。用户安装后不需要额外安装 Homebrew、MKVToolNix、FFmpeg 或 Python。

## 安全提示

此 DMG 未使用 Apple Developer ID 签名，未做 Apple notarization。首次打开时，如果 macOS 拦截：

1. 右键点击 `J-MKV-subMuxer.app`。
2. 选择“打开”。
3. 在系统确认框里再次选择“打开”。

## 当前边界

- 软字幕封装，不做硬字幕烧录。
- 不做视频转码。
- 不做音频转码。
- 不做分辨率调整。
- 不做码率调整。
